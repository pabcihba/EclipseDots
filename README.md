# Default keybinds

`⌘ Super + ENTER ` - Terminal (kitty)

`⌘ Super + R ` - App launcher

`⌘ Super + E ` - File explorer (nemo)


`⌘ Super + Q ` - Close an app

`⌘ Super + T ` - Toggle floating mode

`⌘ Super + (1-9) ` - Focus on (1-9) workspace

`⌘ Super + SHIFT + (1-9) ` - Moves active window in (1-9) workspace

`⌘ Super + SHIFT + (arrow keys) ` - Resizes active window

`⌘ Super + CTRL + (arrow keys) ` - Moves active window

`⌘ Super + (arrow keys)` - Moves focus

`⌘ Super + K` - Swaps windows

`⌘ Super + J` - Splits windows

`⌘ Super + LMB` - Move a window

`⌘ Super + RMB` - Resize a window

`⌘ Super + F` or `F11` - Fullscreen toggle

`⌘ Super + P ` - Toggle pseudo mode

`⌘ Super + H ` - Move in scratchpad

`⌘ Super + D` - Toggle bar

`⌘ Super + C ` - Calculator

`⌘ Super + SHIFT + C ` - Screen recorder

`⌘ Super + M ` - Dashboard

`⌘ Super + N ` - Settings

`⌘ Super + V` - Clipboard manager

`⌘ Super + L` - Session menu

`⌘ Super + SHIFT + L` - Lock screen

`⌘ Super + PrtSc` - Fancy screenshot

`PrtSc` - Full screenshot

`⌘ Super + End` or `CTRL + SHIFT + ESC` - Process manager (btop)

`⌘ Super + SHIFT + N ` - Toggle 'Do Not Disturb' mode

`⌘ Super + CTRL + N ` - Clear all notifications

`⌘ Super + I` - Zooms in

`⌘ Super + O` - Zooms out

`⌘ Super + SHIFT + (mouse scroll)` - Zooms screen

`⌘ Super + (mouse scroll)` - Scrolls existing workspaces

`⌘ Super + CTRL + (mouse scroll)` - Scrolls all workspaces

# Features

- fish, fastfetch and kitty configs
- Wallpaper-based colorschemes for everything 
- Overview feature for better workspace and window management
- Screenshot utility with eye-candy animations
- Vertical workspaces scrolling for extra mouse buttons
- Window bars like in floating window managers
- Screen zooming
- Gamemode
- Keybinds to control your mouse using keyboard
- Some gestures for laptop users

All the features of Noctalia Shell, with is:

- Easily customizable panel
- Lockscreen
- App launcher, widgets for Wi-Fi, Bluetooth, volume/brightness control, calendar, notifications panel
- Settings app
- Stunning dashboard with audiovisualizer and weather effects
- Integration with Walogram for dynamic wallpaper based Telegram themes (optional
- Calculator, emoji picker, clipboard manager
- Power profiles manager
- Screen recorder
- Wallpaper management, you can search and apply wallpapers from your panel
