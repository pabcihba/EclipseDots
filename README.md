# Features

- fish, fastfetch and kitty configs
- Wallpaper-based colorschemes for everything 
- Overview feature for better workspace and window management
- Screenshot utility with eye-candy animations
- Vertical workspaces scrolling with extra mouse buttons
- Window bars like in floating window managers

All the features of Noctalia Shell, with is:

- Easily customizable panel
- Lockscreen
- App launcher, widgets for Wi-Fi, Bluetooth, volume/brightness control, calendar, notifications panel
- Settings app
- Stunning dashboard with audiovisualizer and weather effects
- Integration with Walogram for dynamic wallpaper based Telegram themes (optional
- Calculator, emoji picker, clipboard manager
